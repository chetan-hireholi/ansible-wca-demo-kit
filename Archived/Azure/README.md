## Provision Azure VM

This examples showcases 2 workarounds in provisioning Azure VM:
1. First task does not use the configuration mentioned in the `vars`
2. Second task uses the configuration mentioned in the `vars`

You can find the complete playbook in file `solution_azure_vm_single-task.yml` in the same folder.
